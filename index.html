Welcome to EduGIS at github

You can find source code and (some) documentation for the EduGIS projects here.

Projects

  • EduGIS Viewer
    • Github repository: edugis-viewer
    • Viewer configured for the netherlands
    • Viewer configured for the world
    • Viewer configured for the world (mapbox-gl)
    • See EduGIS configurations below for more regions and themes
  • EduGIS configurations
    • Github repository: edugis-configurations
    • Configurations in the master branch are published via:
      https://edugis-org.github.io/edugis-configurations
      For example:
      • world.json is available at
        https://edugis-org.github.io/edugis-configurations/world.json
        So you can use: #configurl=https://edugis-org.github.io/edugis-configurations/world.json
        to view the world.json configuration in the viewer.
